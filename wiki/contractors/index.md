# Contractors

Profiles of BBMP/GBA contractors — government PSUs, private firms, and individuals — with their history, aliases, blacklist flags, and key projects.

## Flagged Contractors

These contractors have been flagged on one or more debarment lists or documented as involved in civic scams.

- [**KRIDL** — Karnataka Rural Infrastructure Development Limited](kridl.md) — Government PSU, blacklisted twice, received Rs 4,700 Cr via Section 4(g) tender exemption. Under ED scanner (2025).

## How to Add a Contractor

1. Copy [`_template.md`](_template.md) to a new file named after the contractor (e.g., `contractors/acme-builders.md`)
2. Fill in identity, activity, flags, key projects, and sources
3. Submit a PR on GitHub
4. Cite every claim with a verifiable source
