# APSDMA — AP State Disaster Management Authority

Real-time disaster response and weather data. Especially relevant for Vizag (cyclone-prone coastal city).

## Portal

- **Main**: [apsdma.ap.gov.in](https://apsdma.ap.gov.in/)
- **GIS dashboard (live)**: [apsdmagis.ap.gov.in](https://apsdmagis.ap.gov.in/)
- **Twitter**: [@APSDMA](https://twitter.com/APSDMA)

## What's accessible

- Live AWS (Automatic Weather Station) feeds — wind, rainfall, temperature, humidity
- State Emergency Operations Centre status
- Cyclone preparedness and response plans by district
- Real-time alerts via Twitter (highest signal during cyclones)
