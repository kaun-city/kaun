# Partners & Contributors

The Civic Data Commons is built by and with the civic community. No single project could build this alone.

## Founding collaboration

### [BNP (Bengaluru Navanirmana Party)](https://bright.nammabnp.org)

BNP is a civic political party in Bengaluru that has been systematically documenting public money in the city. Key contributions to the commons:

- **B-RIGHT portal** — 63,629 BBMP projects documented with ward-level detail, contractor names, and payment traces. [bright.nammabnp.org](https://bright.nammabnp.org)
- **KRIDL 4(g) investigation** — documented the Rs 4,700 Cr exemption route used to bypass normal tendering. [Finding](../bengaluru/findings/kridl-4g-exemption.md)
- **Pointer to primary sources** — surfaced the 4 government portals that feed the commons (BBMP IFMS, KPPP, BBMP HRMS, KarnatakaTenders)

**Srikanth Narasimhan** from BNP initiated the collaboration after seeing kaun.city on LinkedIn. The vision that emerged from that conversation: instead of each civic tool scraping data into its own silo, build a shared open commons that everyone can contribute to and consume from.

### [kaun.city](https://kaun.city)

The map-first citizen interface. Contributions to the commons:

- **Entity-resolved contractor profiles** — 1,305 contractors unified across aliases, phone numbers, and ward footprints
- **Cross-ward analytics** — spending patterns, blacklist cross-references, WardGrade scoring
- **Public JSON APIs** — four CORS-enabled endpoints serving ward, contractor, rep, and spending data
- **Citizen reporting** — photo-verified ward reports with AI moderation

## Data sources

The commons stands on open data efforts that came before:

- **[opencity.in](https://data.opencity.in)** — the backbone of civic datasets in India
- **[DataMeet](https://github.com/datameet)** — ward boundary GeoJSON for 28 cities
- **[MyNeta / ADR](https://myneta.info)** — candidate affidavits
- **[TCPD Lok Dhaba](https://tcpd.ashoka.edu.in/lok-dhaba)** — election results archive
- **[OpenStreetMap](https://openstreetmap.org)** — amenities, signals, bus stops
- **[LGD](https://lgdirectory.gov.in)** — canonical government entity IDs

## Want to contribute?

We welcome:

- **Investigators & journalists** — document findings with sources
- **Civic activists** — share RTI responses, ground-truth data
- **Engineers** — improve scrapers, build integrations, fix bugs
- **Domain experts** — audit data quality, flag errors
- **Civic tools** — consume the APIs, point your users here
